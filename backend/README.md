# Backend

This is a [Node](https://nodejs.org/en) project using [Fastify](https://fastify.dev/).

## Features

- Interact with the EDC management API to create contract agreements and transfer processes
- Verify signed data request messages
- Verify ownership of agreement NFTs
- Verify contract agreement details
- Fetch requested asset data via internal data API after successful verification

## Prerequisites

- Node v22.19 or higher
